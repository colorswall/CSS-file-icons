CSS file icons 0.1.0

Pure CSS file icons for most popular file extensions and file types.
GitHub Star

How to use

Donwload css-file-icons or install via npm

npm install css-file-icons --save

Include css-file-icons.css
Inside html insert block with class fi: 

<div class="fi fi-doc">
    <div class="fi-content">doc</div>
</div>
To change icon color change class name. For example for EXE file class name fi-exe 
<div class="fi fi-exe">
